# TAZLARM v2.2.1 notification resource fix

GitHub Actions reached Kotlin compilation but failed because `AlarmService.kt` referenced `R.string.alarm_channel_description` while that resource was missing from `strings.xml`.

This build adds the missing channel description, adds a validator assertion for it, verifies that every `R.string` reference resolves, and updates the version and APK artifact name to v2.2.1.
