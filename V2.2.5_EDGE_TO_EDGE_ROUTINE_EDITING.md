# TAZLARM v2.2.5 edge-to-edge and setup refinement

This build makes the live sunrise background fill the complete display, including behind transparent system bars. Content uses safe-area padding, so the header and controls remain clear of cutouts and navigation gestures.

The first-run name field is now deliberately minimal. Onboarding routine cards use pale-blue surfaces and always show an Edit pencil. The same detailed step editor used after setup is available during onboarding, allowing question topics/counts, accepted barcodes and photo references to be configured before setup is finished.
