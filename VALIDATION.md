# TAZLARM v2.3.1 validation

The package validator has been updated for the v2.3.1 version and the new scenic renderer.

It validates:

- versionName 2.3.1 and versionCode 231;
- the night-to-day scenic sunrise renderer;
- mountain, lake and meadow landscape layers;
- the dedicated Alarm Finished screen and both completion actions;
- the slightly lowered live-alarm branding;
- all previously validated alarm, routine, permission, picker and trivia behaviour.

The included GitHub Actions workflow remains the authoritative full Android compilation and APK test.
